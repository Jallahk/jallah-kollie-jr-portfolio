# Jallah Kollie Jr Portfolio

This project is my personal portfolio website, built to showcase my Python and software development work in a way that is easy to share with employers, professors, and collaborators. I created it as a central place to highlight projects from my Machine Learning and A&D coursework, along with private demo space for projects that are still in progress.

The site organizes my work into separate tabs so visitors can explore my Machine Learning projects, my Algorithms and Data Structures work, and my Lovable project demo section. Each project card includes a short description, and when a project is selected, it opens a popup with more information and a preview of the source code. I also added pagination so the projects load in smaller groups instead of all at once.

One of the main goals of this portfolio was to present real work from my own folders in a clean and professional format. To do that, I used a Python generator script to scan my project files, collect metadata, and build the project catalog that powers the website. That lets me keep the portfolio connected to the work I have actually completed instead of manually rewriting every project from scratch each time.

I also used AI-assisted development and prompt engineering with OpenAI Codex while building and improving this site. I used structured prompts to help design features, debug issues, improve the user interface, and organize how the portfolio content is displayed.

This portfolio is published as a static website and is intended to be updated over time as I continue building new projects and refining the ones already included here.
